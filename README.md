# 10x Chat - Web App (Next.js)

## Setup

1. Install:
```
npm install
```

2. Edit `.env.local` with your backend URL

3. Dev:
```
npm run dev
```

4. Production build:
```
npm run build
npm start
```

## Deploy on Hostinger
- Upload built files or use Node.js hosting
- Point `chat.10xdigitalventures.com` to this app
- Backend runs on port 5000, this on port 3000
- Use Nginx to route accordingly
